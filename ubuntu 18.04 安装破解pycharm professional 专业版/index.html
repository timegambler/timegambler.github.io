ubuntu 18.04 安装破解pycharm professional 专业版

第一步，下载（不得不说速度很慢）
官网地址：http://www.jetbrains.com/pycharm/?fromMenu
第二步，安装
安装的话，只需要解压就行
（1）进入你的pycharm-xxx-2019.2.3.tar.gz文件所在的目录

cd XXX

（2）直接用解压缩就行

tar -zxvf pycharm-xxx-2019.2.3.tar.gz

（3）可以移动到opt目录（这一步随意，我移动过去了）
sudo mv pycharm-XXX/ /opt
（4）启动一下

sudo sh /opt/pycharm-XXX/bin/pycharm.sh

点击试用就行，不用管其他的，这里没什么需要操作的
基本算是安装成功了
第三步，破解（重点啊！！！我试过很多方法，目前只有这个可行）
破解需要一个文件：jetbrains-agent.jar ，文件链接：https://pan.baidu.com/s/1Z89HKV1VidyZHY0BnOWP7w 提取码：bn7h
下载完成后移动到刚刚解压的在/opt目录的pycharm/bin目录中

sudo mv jetbrains-agent.jar  /opt/pycharm-XXX/bin

重点来了！！！

cd /opt/pycharm-XXX/bin 先进入pycharm的bin目录中ll

会看到绿框中的两个文件：pycharm64.vmoptions和pycharm.vmoptions，修改这两个文件

![](/ubuntu 18.04 安装破解pycharm professional 专业版/1_ubuntu 18.04 安装破解pycharm professional 专业版.png)

在这两个文件的最后一行均加上一行内容路径并保存：

sudo gedit pycharm.vmoptions
#最后一行 加上-javaagent：/opt/pycharm-XXX/bin/jarbrains-agent.jar
sudo gedit pycharm64.vmoptions#同上

-javaagent：/opt/pycharm-XXX/bin/jarbrains-agent.jar( 就是刚刚下载的jar文件路径)
保存成功后，重新启动

sudo sh pycharm.sh

![](/ubuntu 18.04 安装破解pycharm professional 专业版/2_ubuntu 18.04 安装破解pycharm professional 专业版.png)
